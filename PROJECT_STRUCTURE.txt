Algorithm Flow:
===============

Input Data
    |
    v
InputCharacteristics Analysis
    ├── Presortedness: 0.0 to 1.0
    ├── Range Density: 0.0 to 1.0
    ├── Distribution Type: uniform/normal/skewed
    ├── Has Duplicates: bool
    └── Data Range: (min, max)
    |
    v
Strategy Selection
    ├── Size <= 20? ──────────────> InsertionSort
    ├── Presortedness >= 0.7? ────> InsertionSort
    ├── Dense Range + Positive? ──> RadixSort
    └── Default ──────────────────> MergeSort
    |
    v
Adaptive Sorting
    ├── Recursive division
    ├── Re-analyze subsections
    ├── Switch strategies as needed
    └── Track all decisions
    |
    v
Output
    ├── Sorted array
    └── Statistics
        ├── Comparisons
        ├── Swaps
        ├── Execution time
        └── Strategy switches

Usage Example:
==============

from smart_sort import SmartSort

# Basic usage
data = [64, 34, 25, 12, 22, 11, 90]
sorter = SmartSort()
result = sorter.sort(data)

# With statistics
sorter = SmartSort(verbose=True)
result = sorter.sort(data)
stats = sorter.get_stats()

print(f"Time: {stats['execution_time']*1000:.4f} ms")
print(f"Comparisons: {stats['comparisons']}")
print(f"Strategy: {stats['strategy_switches'][0]['strategy']}")
