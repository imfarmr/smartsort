## Font Information

The website uses Geist and Geist Mono fonts. If font files are not available in the `media/` directory, the site automatically falls back to system fonts with adjusted metrics for similar appearance.

## Customization

### Change Colors
Edit CSS variables in `css/styles.css`:
```css
:root {
    --accent-primary: #0070f3;  /* Change accent color */
    --bg-primary: #000000;       /* Change background */
}
```

### Modify Content
Edit sections directly in `index.html`

### Add Features
Extend functionality in `js/main.js`
