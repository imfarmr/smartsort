Quick Start

Basic Usage

from smart_sort import SmartSort

data = [64, 34, 25, 12, 22, 11, 90]
sorter = SmartSort()
sorted_data = sorter.sort(data)
print(sorted_data)

With Verbose Output

sorter = SmartSort(verbose=True)
sorted_data = sorter.sort(data)

Get Statistics

stats = sorter.get_stats()
print(f"Time: {stats['execution_time']*1000:.4f} ms")
print(f"Comparisons: {stats['comparisons']}")
print(f"Swaps: {stats['swaps']}")

Analyze input data characteristics.

Attributes:

  • presortedness: 0.0 (reverse) to 1.0 (sorted)
  • range_density: 0.0 (sparse) to 1.0 (dense)

  • has_duplicates: Boolean
  • data_range: (min, max) tuple

Sorting Algorithms

InsertionSort

Used for: Small arrays (â‰¤20 elements), nearly sorted data (â‰¥70% presorted)

Time Complexity: O(nÂ²) worst case, O(n) best case

Space Complexity: O(1)

Best for: Nearly sorted data, small datasets

MergeSort

Used for: Large random arrays, general-purpose sorting

Time Complexity: O(n log n) guaranteed

Space Complexity: O(n)

Best for: Random data, guaranteed performance

RadixSort

Used for: Dense integer ranges with good range density

Time Complexity: O(dÂ·n) where d is number of digits

Space Complexity: O(n + k)

Best for: Dense non-negative integer ranges

Usage Examples

Example 1: Nearly Sorted Data

data = list(range(1, 101))
for _ in range(5):
    i, j = random.randint(0, 99), random.randint(0, 99)
    data[i], data[j] = data[j], data[i]

sorter = SmartSort(verbose=True)
result = sorter.sort(data)

# Expected: InsertionSort selected, O(n) performance

Example 2: Dense Integer Range

ages = [random.randint(0, 120) for _ in range(200)]

sorter = SmartSort()
sorted_ages = sorter.sort(ages)

# Expected: RadixSort selected for dense range

Example 3: Random Data

data = [random.randint(1, 100000) for _ in range(1000)]

sorter = SmartSort()
result = sorter.sort(data)

# Expected: MergeSort with InsertionSort for small segments

Example 4: Analyze Before Sorting

from smart_sort import InputCharacteristics

chars = InputCharacteristics(data)
print(f"Presortedness: {chars.presortedness:.2%}")
print(f"Range Density: {chars.range_density:.2%}")
print(f"Distribution: {chars.distribution_type}")
